# 📊 Array Methods Project (JavaScript)

This is a simple JavaScript project that demonstrates all important **Array Methods** using a user-friendly UI.  
Users can add data and perform different array operations like push, pop, map, filter, reduce, etc.

---

## 🚀 Features

- ➕ Add User (push)
- ❌ Remove Last User (pop)
- ❌ Remove First User (shift)
- 🔼 Add First User (unshift)
- ✂️ Delete specific item (splice)
- 📄 Copy data (slice)
- 🔗 Merge arrays (concat)
- 🔁 Reverse array
- 🔢 Sort by age
- 🧠 Map (transform data)
- 🔍 Filter (18+ users)
- ➗ Reduce (total age)
- 🔎 Find user
- ✔️ Some / Every check
- 📌 Includes / IndexOf
- 🔗 Join / ToString
- 📉 Flat
- ✂️ Split string

---

## 📸 How It Works

1. Enter user name and age  
2. Click **Add User (push)**  
3. Data is stored in an array  
4. Click buttons to perform different array methods  
5. Output is shown in card format or alert

---

## ⚙️ Example Output

```
0 - Ankit (20)
1 - Rahul (15)
2 - Neha (25)
```

After filter (18+):

```
0 - Ankit (20)
1 - Neha (25)
```
